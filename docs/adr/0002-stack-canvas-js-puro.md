# Stack: HTML5 Canvas + JavaScript puro, sem engine

Consideramos Phaser/Kaboom.js (engine 2D em JS) e Godot (engine dedicada com editor visual) como alternativas. Optamos por Canvas + JavaScript puro pelo escopo pequeno e didático do projeto, evitando dependência externa e a curva de aprendizado da API de uma engine. Trade-off aceito: sem física, gerenciamento de sprites ou loop de jogo prontos — tudo isso é escrito à mão.
