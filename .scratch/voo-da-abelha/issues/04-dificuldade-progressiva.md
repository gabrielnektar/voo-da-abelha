# 04 — Dificuldade progressiva

**What to build:** A velocidade de rolagem (movimento da abelha e das barreiras) aumenta gradualmente ao longo de cada run, tornando os desvios mais difíceis quanto mais tempo/distância a run dura. No início de cada nova run, a velocidade volta à velocidade base.

**Blocked by:** 02 — Barreiras

**Status:** ready-for-agent

- [ ] A velocidade de rolagem no início de cada run é sempre a mesma velocidade base
- [ ] A velocidade de rolagem aumenta gradualmente conforme o tempo/distância avança dentro da run atual
- [ ] O aumento de velocidade é visível tanto no avanço da abelha quanto no movimento das barreiras
- [ ] Testes automatizados cobrem o aumento gradual da velocidade de rolagem ao longo do tempo no módulo puro de lógica
