# 03 — Pontuação contínua

**What to build:** A pontuação sobe continuamente conforme a distância percorrida durante a run, exibida em tempo real na tela enquanto o jogador joga. Ela reseta para zero no início de cada nova run e aparece como pontuação final na tela de game over.

**Blocked by:** 01 — Loop básico jogável

**Status:** ready-for-agent

- [ ] A pontuação aparece na tela durante o jogo e sobe continuamente enquanto a run está ativa
- [ ] A pontuação cresce proporcionalmente à distância percorrida, não a barreiras ultrapassadas
- [ ] A pontuação reseta para zero no início de cada nova run
- [ ] A tela de game over mostra a pontuação final da run que acabou de terminar
- [ ] Testes automatizados cobrem o acúmulo contínuo de pontuação no módulo puro de lógica
